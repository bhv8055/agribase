# **App Name**: AgriAssist AI

## Core Features:

- Crop Recommendation: Predict the best crop to grow based on soil and weather inputs using a pre-trained ML model tool.
- Fertilizer Recommendation: Suggest the appropriate fertilizer type and amount based on soil type and crop selection using a pre-trained ML model tool.
- Crop Disease Detection: Detect crop diseases from uploaded leaf images using a deep learning model tool.
- Animal Disease Prediction: Predict animal diseases from uploaded images using a deep learning model tool.
- Instructional Guidance: Provide clear, step-by-step instructions for using each feature, ensuring ease of use for all users.
- Streamlit Interface: Develop a multi-page Streamlit application for user interaction.
- Model Loading & Handling: Load pre-trained ML and DL models to give agriculture and animal recommendations.

## Style Guidelines:

- Primary color: Earthy green (#38A3A5) to represent agriculture and nature.
- Background color: Light beige (#F5F5DC) for a natural and calming feel.
- Accent color: Warm orange (#E56B6F) to highlight important information and CTAs.
- Body font: 'PT Sans', a humanist sans-serif, will be used for the body, and 'Space Grotesk' for headings.
- Employ a clean, intuitive layout with Streamlit's wide layout to maximize screen real estate. Sidebar navigation for easy access to different modules.
- Use clear, representative icons for navigation and feature descriptions.
- Implement subtle animations for feedback, such as loading indicators or success confirmations.
